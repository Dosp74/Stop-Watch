# ⏰ Stop-Watch

## 📋 주요 기능
1. **스톱워치** - 시간을 측정합니다.  
2. **기록 추가** - 측정된 시간을 구간 기록에 추가합니다.  
3. **선택 삭제** - 특정 기록을 선택하여 삭제합니다.  
4. **전체 선택** - 모든 기록을 선택합니다.  
5. **전체 삭제** - 모든 기록을 삭제합니다.

### 🔧 사용 방법
1. **START** 버튼을 클릭하여 시간을 측정하세요.
2. 원하는 시간 기록을 **추가** 및 **관리**하세요.
3. 불필요한 기록은 **삭제**하여 정리하세요.
